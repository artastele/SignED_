## 🎯 HOW TO USE ON NEW PC

### Step 1: Copy Files
Copy entire `SignED_/` folder to new PC

### Step 2: Setup Database
1. Open phpMyAdmin
2. Run: **DATABASE_MASTER_SETUP.sql**
3. Done! ✅

### Step 3: Configure
1. Update `config/config.php`:
   - Database credentials
   - SMTP credentials
2. Login as admin: `admin@signed.local` / `password`
3. Change password
4. Configure SMTP in Admin > Settings

### Step 4: Ready!
System is ready to use! ✅
